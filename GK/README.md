# CLAT General Knowledge & Current Affairs (GK) Module

Welcome to the **CLAT GK & Current Affairs Module** directory!

---

## 📂 Folder Structure

- **`data/`**: Dump your GK PDF source documents, Excel spreadsheets, monthly current affairs roundups, and static question banks here.
- **`src/`**: Source code for the interactive GK Quiz & Current Affairs Recall Engine.

---

## 📥 Instructions for Dumping Files

You can copy and dump all your GK & Current Affairs files into this directory:
- PDF Chapters (e.g., `GK_Current_Affairs_2025.pdf`, `Static_GK_Law.pdf`)
- Excel Question Banks (`GK_31Day_Drill_Schedule.xlsx`)
- Text & JSON Datasets
